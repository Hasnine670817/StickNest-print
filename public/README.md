# Public Assets Folder

This folder is used to serve static assets directly. Any file placed here will be accessible at the root URL of your website.

For the **About** page, please upload your images here and rename them to match the following filenames:

1. `about-hero.jpg` (The top orange banner image)
2. `about-laptop.png` (The laptop and stickers image)
3. `about-map.png` (The world map with remote team locations)
4. `about-logos.png` (The customer logos grid)

Once uploaded, they will automatically appear on the About page.
